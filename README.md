This repository contains data structure implementations in Python and corresponding unit tests.



Linked List methods:

    insert()

    pop()

    size()

    search(value)

    remove(node)

    remove_by_value(value)

    contains(value)



Stack methods:

    push(value)

    pop()

    size()


Queue is implemented as a doubly linked list.

Queue methods:

    enqueue(value)

    dequeue()

    size()


Hash Table methods:

    set(key, value)

    get(key)

    hash(key)
